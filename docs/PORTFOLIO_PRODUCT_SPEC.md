# AI Engineer Personal Brand Website
Version: 1.0

---

# Mission

Build a world-class AI Engineer personal brand website that represents a senior-level GenAI engineer rather than a traditional software engineer or student portfolio.

The website should communicate:

• Engineering Excellence
• AI Expertise
• Product Thinking
• Modern Design
• Trust
• Simplicity
• Technical Depth

Within the first 5 seconds, visitors should understand:

"I build production-grade AI systems."
